# Story Generator for Kids

## 📌 Overview
This project is an AI-based Story Generator designed for kids aged 1–12 years. 
It generates interactive and engaging stories based on age criteria to enhance learning and creativity.

## 🚀 Features
- Age-based story generation
- Interactive and simple UI
- AI-powered content generation
- Improves children's learning and imagination

## 🛠️ Tech Stack
- Java (Spring Boot)
- HTML, CSS, JavaScript
- OpenAI API (planned integration)

## 📂 Project Structure
- src/main/java → Backend logic
- src/main/resources → Configuration files
- pom.xml → Project dependencies

## 🔮 Future Enhancements
- Multi-language support (Hindi & English)
- OpenAI API integration
- More interactive UI
